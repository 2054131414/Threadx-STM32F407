# STM32F407VGT6

#### 介绍
*********************************************工程简介*********************************************

1. 基于STM32F407的物联网网关,采用LoRa,RS485两种通讯方式进行数据数据采集,通过以太网连接服务器上传至云平台;
2. 代码托管至GitHub
3. 开发环境：ARM MDK Keil
4. 软件平台：ST(意法半导体)HAL库 + AzureRTOS
5. 编译器：arm clang(AC6)
6. 硬件平台：STM32F407VGT6

#### 软件架构
整体原则：上下分离，左右分离，不通层级之间禁止跨层调用
